Como funciona

Senhas são geradas localmente no seu navegador usando aleatoriedade criptográfica (Web Crypto API), sem envio de dados para nenhum servidor. Ideal para redes sociais, bancos e sistemas em geral.


Dica: use senhas diferentes para cada serviço e guarde-as em um gerenciador de senhas confiável, em vez de reutilizar a mesma senha em vários lugares.
