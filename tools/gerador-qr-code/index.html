Como funciona

Links e URLs

Use a URL completa, incluindo o protocolo https://. Isso garante que o QR Code redirecione corretamente ao ser escaneado.

Pagamentos PIX

Cole o código "Pix copia e cola" gerado pelo seu banco ou aplicativo financeiro. O QR Code resultante pode ser escaneado por qualquer app de pagamento.

Níveis de correção de erro

Nível Recuperação Ideal para
L ~7% Telas digitais, URLs longas
M ~15% Uso geral (recomendado)
Q ~25% Impressão em papel
H ~30% Adesivos, superfícies com desgaste

Dicas de uso

  • Sempre teste o QR Code escaneando antes de distribuir ou imprimir.
  • Para impressão, prefira tamanho Grande ou Extra grande e correção Q ou H.
  • Mantenha bom contraste entre as cores dos módulos e do fundo para garantir leitura.
  • Cores personalizadas podem ser menos legíveis em câmeras antigas — prefira preto sobre branco em casos críticos.
  • O processamento é feito no seu navegador — nenhum dado é enviado a servidores.
