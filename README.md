# 🚀 UtiliWeb

**UtiliWeb** é uma plataforma de ferramentas online gratuitas para uso rápido no navegador.

## 🌐 Acesso
https://utiliweb.vercel.app/

## 🧠 Ferramentas
- Contador de Caracteres
- Calculadora de Porcentagem
- Gerador de Senhas
- Conversor de Temperatura

## ⚡ Objetivo
Criar um conjunto de ferramentas rápidas, leves e úteis para o dia a dia.

## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript
- Vercel (deploy)
- GitHub (versionamento)

## 📈 Status
Em desenvolvimento contínuo
